# 🏠 체크마이룸 (Check My Room)

> **"사진 한 장으로 끝내는 내 방 안전 진단 & 집주인 소통 가이드"**
>
> 사회초년생의 부동산 장벽을 AI로 해결하는 MVP 서비스입니다.

<br>

## 🚨 문제 정의 (Problem)

단순히 "자취생이 방 구하기 어렵다"는 것이 아닙니다. 청년들이 겪는 **진짜 문제**는 다음과 같습니다.

1. **법률 용어 해독 불가:** 등기부등본을 발급받아도 '근저당권', '선순위 채권' 등의 용어를 이해하지 못해 서류를 보고도 사기 위험에 노출됩니다.
2. **소통의 두려움:** 방의 하자를 발견해도 정확한 명칭(예: 강마루 파손, 결로 현상 등)을 몰라 제대로 따지지 못하며, 임대인에게 수리를 요청하는 것 자체에 심리적 장벽을 느낍니다.

## 💡 해결 방안 (Solution)

**"AI를 통한 부동산 용어 번역 및 커뮤니케이션 대행"**

사용자가 눈으로 본 직관적인 이미지(서류, 방 사진)를 업로드하면, AI(Gemini)가 이를 전문적인 부동산/법률 데이터로 분석하고 **행동 지침(문자 템플릿, 대처 매뉴얼)**으로 변환하여 제공합니다.

<br>

## ✨ 핵심 기능 (Core Features)

### 1️⃣ 계약 전 체크리스트
- 전세/월세 및 건물 유형에 따라 반드시 확인해야 할 맞춤형 체크리스트 제공
- 진행 상황을 브라우저 localStorage에 저장하여 새로고침 후에도 체크 이력 유지

### 2️⃣ 방 상태 분석 및 집주인 소통 봇
- **다중 이미지 종합 분석:** 여러 부위 사진을 한 번에 올리면 AI가 모든 하자를 통합 분석
- **AI 하자 분석:** 사진 속 하자를 '건축/부동산 전문 용어'로 정확히 진단
- **문자 템플릿 자동 생성:** 수리 요청 문자를 즉시 복사 가능한 형태로 제공
- **평균 수리 견적 제공:** 발견된 하자 종류에 따른 시장 평균 수리 비용을 참고용으로 안내
- **수리 업체 연결:** 네이버 지도 '주변 인테리어 수리' 검색으로 바로 연결되는 외부 링크 버튼 제공

### 3️⃣ 등기부등본 위험도 스캐너
- **다중 페이지 업로드:** 갑구·을구 등 여러 페이지를 한 번에 올려 종합 분석
- **위험도 점수화:** 안전(🟢 0-30점), 주의(🟡 31-70점), 위험(🔴 71-100점) 3단계 게이지 시각화
- **6대 핵심 항목 체크포인트:** 근저당/담보권, 소유권 안전성, 압류, 신탁등기 등 핵심 요소를 아코디언 형태로 상세 분석
- **법률 용어 툴팁:** 결과 텍스트 내 어려운 법률 용어를 터치하면 쉬운 설명 말풍선 표시
- **조건부 대처 및 BM 연동 (스마트 라우팅):** - 위험(🔴) 판정 시: HUG 안심전세포털 대처 매뉴얼 링크 자동 노출
  - 안전(🟢) 판정 시: 1인 가구 이사/입주청소 플랫폼(짐싸) 제휴 링크 노출로 수익 모델(BM) 확보

### 4️⃣ 최근 분석한 내 방 기록 (보관함)
- **비교 분석 지원:** 여러 방을 보러 다니는 유저를 위해 분석 결과를 localStorage에 자동 저장
- **직관적인 네이밍:** 사용자가 직접 방 이름(예: 신림역 투룸)을 지정하여 저장하고 메인 화면에서 한눈에 비교 가능

## 🚀 프로젝트 실행 방법 (How to run)

```bash
# 1. 저장소 클론
git clone https://github.com/wrtn-edu-sch-bootcamp/20231456-Personal-Project.git

# 2. 패키지 설치
npm install

# 3. 환경 변수 설정
# 루트 경로에 .env.local 파일 생성 후 아래 내용 입력
GEMINI_API_KEY="본인의 Gemini API 키 입력"

# 4. 개발 서버 실행
npm run dev
```

> **API 키 발급:** [Google AI Studio](https://aistudio.google.com/app/apikey)에서 무료로 발급받을 수 있습니다.
